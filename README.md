# Wenex (위넥스) ✈️

**여행의 순간이 지도가 되다**

Wenex는 여행지에서 촬영한 사진을 통해 자동으로 지역을 인증하고, 나만의 디지털 지도를 만들어주는 새로운 형태의 여행 기록 서비스입니다.

![Wenex Landing Page](./assets/wenex_login.png)

## 🌟 주요 기능

*   **📸 사진 기반 지역 인증**: 복잡한 절차 없이 사진 한 장으로 위치를 인증하세요.
*   **🗺️ 디지털 커뮤니티 맵**: 나의 여행 발자취를 지도에 남기고 다른 여행자들과 공유하세요.
*   **🎟️ 실시간 쿠폰 발행**: 인증된 장소 주변의 맛집, 카페 쿠폰을 실시간으로 받아보세요.

## 🛠️ 기술 스택

*   **HTML5**: 시맨틱 마크업 구조
*   **CSS3**: Modern CSS, Flexbox/Grid, Animations, Glassmorphism Design
*   **Font**: Pretendard (CDN)

## 🚀 실행 방법

1. 저장소를 클론합니다.
   ```bash
   git clone https://github.com/ted-dylan/wenex.git
   ```
2. `index.html` 파일을 브라우저에서 엽니다.
   또는 VS Code의 Live Server 등을 이용하여 실행합니다.
